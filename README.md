# léalivre

## Description
Ce projet est un site web de catalogue de livres, permettant aux utilisateurs de parcourir et d'acheter des livres en plusieurs langues, incluant des livres en français, en anglais et en arabe.

## Fonctionnalités
Navigation facile : Parcourez les livres par langue et par catégorie.
Système de paiement : Achat sécurisé en ligne.
Recherche avancée : Trouvez des livres spécifiques rapidement.
Interface utilisateur attrayante : Design moderne et responsive.

## Technologies utilisées
HTML/CSS
JavaScript
PHP
MySQL (pour la base de données)
